YesNoPillow
=======

結婚記念アプリ。

NFCタグに Yes or No を書き込んだり、読み込むだけのアプリ。


## アプリの概要

- NFCタグにテキストデータ（「YES」または「NO」）を読み書きする。

- 読み込んだテキストデータによって、表示する画像が変わる。

- 対応フォーマット外のNFCは、NFCのIDのみを表示する。


## クラスの説明

- ReadActivity.java

    NFCタグを読み込む画面クラス。

- WriteActivity.java

    NFCタグに書き込む画面クラス。

- AbstractActivity.java

    上記2クラスの基底クラス。
